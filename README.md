# nat-device-vibration

## Installation
```
weexpack plugin add nat-device-vibration
```

```
npm install weex-nat --save
```

## Usage

Use in weex file (.we)

```html
<script>
import 'Nat' from 'weex-nat'

// vibrate
Nat.vibrate()

</script>
```

See the Nat [Documentation](http://natjs.com/) for more details.
